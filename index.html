Questions fréquentes

FAQ

Shiva OS est-il compatible avec les GPU AMD et NVIDIA ? ▼
Oui ! Shiva OS inclut les drivers Mesa à jour pour AMD (RDNA 1/2/3) et Intel Arc. Pour NVIDIA, les drivers propriétaires sont disponibles via le gestionnaire de pilotes. Les deux sont testés et validés.
Quelle est la différence avec Ubuntu ou Kubuntu classique ? ▼
Shiva OS est une surcouche gaming complète : noyau temps réel, GameMode, MangoHUD, Proton, Lutris et Bottles sont préconfigurés dès l'installation. Zéro configuration manuelle, 100% prêt à jouer.
Puis-je jouer à des jeux Windows sous Shiva OS ? ▼
Absolument. Proton (Steam), Wine (Bottles/Lutris) et Heroic pour les jeux Epic/GOG sont tous préinstallés. La majorité des jeux AAA Windows fonctionnent nativement. Consultez ProtonDB pour la compatibilité par jeu.
Quelles sont les configurations minimales requises ? ▼
CPU : x86_64 double cœur (Intel Core 2 Duo ou AMD Athlon 64 X2 minimum). RAM : 4 Go minimum, 8 Go recommandé. Stockage : 25 Go minimum. GPU : Compatible Vulkan recommandé pour les jeux AAA.
Shiva OS reçoit-il des mises à jour régulières ? ▼
Oui, chaque semaine. Basé sur Kubuntu 26.04 LTS (5 ans de support), Shiva OS reçoit les mises à jour de sécurité Ubuntu + ses propres optimisations gaming via le dépôt PPA officiel ppa:shiva-os/purity.

Aide, guides et assistance technique pour Shiva OS.

🔱 Comment installer Shiva OS ? ▼
Téléchargez l'ISO (disponible le 24 avril), flashez-la sur une clé USB avec BalenaEtcher ou Rufus, et bootez dessus. L'installateur Calamares vous guidera étape par étape.
🎮 Optimiser NVIDIA pour Shiva OS 26.04 ▼
Shiva OS utilise le driver propriétaire 560+ par défaut. Pour vérifier ou changer : ouvrez le "Gestionnaire de Pilotes" dans les paramètres système. Activez le "Full Composition Pipeline" pour éviter le tearing.
🛡️ Anti-Cheat : Pourquoi ça bloque ? ▼
Certains jeux (Call of Duty, Valorant) utilisent des Anti-Cheats au niveau noyau (Kernel) qui ne sont pas compatibles avec Linux. Nous travaillons avec la communauté pour améliorer ce support, mais pour l'instant, ces jeux sont bloqués.
⚡ Activer le Mode eSport (Noyau RT) ▼
Le noyau Temps Réel (RT) est actif par défaut sur Shiva OS. Il garantit que votre jeu a toujours la priorité absolue sur les ressources CPU. Pas besoin de configuration supplémentaire.
